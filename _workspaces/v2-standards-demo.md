This demo workspace showcases the **V2 Blockly Interface Standards**:
1. **Value Block Architecture**: Creation blocks (Ground, Light, Camera, Primitives) return references and are assigned to variables.
2. **Consolidated Blocks**: The `create_primitive` and `create_light` blocks use dropdowns to select types.
3. **Dynamic Inputs**: All coordinates and properties use value sockets, allowing for math and variable inputs.
4. **Clean IDs**: The engine automatically manages internal unique IDs (prefixed with `id_`).
5. **Animation Standards**: Uses the consolidated `animate_transform` block.
